# SAP BTP - Cloud Application Programming Model (CAPM) with Node.js

## 45-Day Training Program | Fresher to Developer

A comprehensive upskilling program covering SAP BTP and the CAP framework, designed for freshers with no prior coding experience.



git remote set-url origin https://btpchamp:@github.com/btpchamp/BTP-May.git


---

## Program Overview

| Detail | Info |
|--------|------|
| **Duration** | 45 Days (8 hours/day) |
| **Target Audience** | Freshers |
| **Daily Format** | Theory (3 hrs) + Hands-on (3.5 hrs) + Assessment (0.5 hr) |
| **End Goal** | Build & deploy a full-stack SAP BTP application |

---

## Weekly Curriculum

| Week | Topics | Days |
|------|--------|------|
| **Week 1** | SAP BTP Foundations & Cloud Concepts | Days 1-5 |
| **Week 2** | JavaScript & Node.js Fundamentals | Days 6-10 |
| **Week 3** | Introduction to CAP & Database Layer (CDS Modeling) | Days 11-15 |
| **Week 4** | Service Layer & OData | Days 16-20 |
| **Week 5** | SAP Fiori UI Development | Days 21-25 |
| **Week 6** | Git, HANA Cloud & Security | Days 26-30 |
| **Week 7** | Deployment to Cloud Foundry | Days 31-35 |
| **Week 8** | Integration, Joule AI & Advanced Topics | Days 36-40 |
| **Week 9** | SAP Build Work Zone, Capstone Project & Final Assessment | Days 41-45 |

---

## Technologies Covered

SAP BTP • SAP CAP (Node.js) • SAP HANA Cloud • SAP Fiori Elements • Cloud Foundry • XSUAA • OData V4 • JavaScript/ES6+ • Git & GitHub • SAP Build Work Zone • SAP Joule • SAP Build Process Automation
